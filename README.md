## 專案介紹
### 專案題目說明
* 題目：

  永豐金控一---千人千面的證券自動化服務
  
* 題目說明：

  傳統投資資訊通常採用多位研究分析人力來處理總經、產業、個股訊息，製成統一的研究報告來服務客戶，並沒有從客戶自身的投資知識圖譜、交易的習慣、投資目標等多方面去客製化服務。在FINTECH時代，我們藉由大數據分析能更了解客戶目前的樣態，並可以提供多樣化的投資資訊來協助客戶找到屬於自己的投資節奏與適合的內容。真正的與客戶個人長期投資成長站在一起。
